새로운 정보(글) 추가 방법

새로운 글을 웹사이트에 추가하는 과정은 크게 2단계로 이루어집니다. 아래의 안내를 따라 진행해 주세요.

1단계: 정보 파일(마크다운) 생성

먼저, 웹사이트에 표시될 실제 내용이 담긴 파일을 만듭니다.

  1. 프로젝트 폴더에서 app/public/articles 폴더로 이동합니다.
  2. 해당하는 주제의 폴더(예: growth, monetization)를 선택합니다.
  3. 그 안에 새로운 텍스트 파일을 만들고, 파일 이름을 my-new-post.md와 같이 영어로 설정합니다. (주의: 파일 확장자는 반드시 .md여야 합니다)
  4. 생성된 파일을 열어, 원하는 내용을 자유롭게 작성합니다. (마크다운 문법을 사용할 수 있습니다)

2단계: 웹사이트 목록에 새 글 등록

파일을 만든 후, 웹사이트가 이 파일의 존재를 알 수 있도록 목록에 등록해야 합니다.

  1. 프로젝트 폴더에서 app/src/pages/CategoryPage.js 파일을 텍스트 편집기로 엽니다.
  2. 파일의 맨 위쪽에 있는 articles 라는 부분을 찾습니다.
  3. 1단계에서 파일을 추가했던 카테고리(예: growth)에, 아래와 같이 새로운 한 줄을 추가합니다.
const articles = {
  growth: [
    { id: 'strategy-1', title: '유튜브 채널 성장을 위한 첫 번째 전략' },
    { id: 'tags-for-views', title: '조회수를 높이는 태그 작성법' },
    { id: 'my-new-post', title: '내가 새로 작성한 글 제목' } // <-- 이 줄을 추가
  ],
  monetization: [
    // ...
  ],
  // ...
};
매우 중요: id는 1단계에서 만들었던 파일의 이름(.md 제외)과 정확히 일치해야 합니다. title은 웹사이트 목록에 표시될 제목입니다.

파일을 저장하면, 웹사이트가 자동으로 새로고침되면서 새로운 글이 목록에 나타납니다. 별도의 저장 버튼 없이, 파일 수정만으로 모든 것이 관리됩니다.
